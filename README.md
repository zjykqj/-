# 车路协同场景下交叉口拥堵预测与小样本迁移控制

## 一、项目简介

本项目围绕**车路协同场景下的城市交叉口拥堵预测问题**展开，重点研究在目标交叉口样本较少的情况下，如何利用多源交叉口数据学习交通运行的共性规律，并通过小样本迁移方法提升目标交叉口的拥堵预测效果。

项目整体流程分为 6 个步骤：

1. 构建交叉口时空样本数据；
2. 训练源域共性拥堵预测模型；
3. 对预测残差进行分解；
4. 构建目标域交叉口嵌入表示；
5. 基于小样本数据拟合目标域个性化修正项；
6. 对迁移预测效果进行评估与可视化。

本项目适用于课程汇报、科研训练、交通流预测实验、小样本迁移学习方法展示等场景。

---

## 二、研究背景

在城市交通系统中，不同交叉口之间既存在相似的交通运行规律，也存在由路网结构、信号配时、周边功能区、交通需求差异等因素导致的个性化特征。

传统预测模型通常需要大量目标交叉口历史数据，但在实际车路协同应用中，部分交叉口可能存在以下问题：

- 新部署路侧设备，历史数据不足；
- 数据质量不稳定；
- 标注样本较少；
- 不同交叉口之间分布差异明显。

因此，本项目采用“**共性学习 + 个性残差修正 + 小样本迁移**”的技术思路，在源交叉口数据较充分、目标交叉口样本较少的条件下，提高目标交叉口的拥堵预测能力。
